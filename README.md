# aws-cloud-security-labs

This repository documents hands-on AWS cloud security labs covering identity and access management (IAM), account hardening, least-privilege access, and core security best practices.

## Topics covered
- AWS account hardening
- IAM users, groups, roles, and policies
- Least-privilege access design
- Multi-factor authentication (MFA)
- Security governance fundamentals

## Structure
Each lab is organized in its own directory and includes:
- Objective
- Actions performed
- Security rationale
- Screenshots and/or diagrams

## Labs
Additional labs will be added as new AWS security topics are explored.
